# 🏛️ Atelier AI — Workplace Productivity Assistant

### 🌟 Project Overview
Atelier AI is an elegant, single-platform workspace assistant designed to automate daily professional tasks. Built as a high-end luxury digital suite, the platform bridges a premium user experience with powerful workplace automation to handle communication, organization, and technical research seamlessly.

---

### 🚀 Key Features Fully Implemented
*   ✉️ **Smart Email Generator:** Composes tailored text block correspondence across multiple selectable professional tones.
*   📝 **Meeting Notes Summarizer:** Distills long transcripts into distinct executive summaries, key decisions, and deadline tracking.
*   📅 **AI Task Planner:** Transforms random lists into clean, interactive, chronological daily agendas.
*   🔍 **AI Research Assistant:** Moves from complex questions to strategic, boardroom-ready topic analysis.
*   🤖 **AI Concierge (Chatbot):** A conversational workspace partner for interactive, real-time queries.

---

### 🛠️ Tools & Tech Stack Used
*   🎨 **Frontend Architecture & Generation:** Lovable AI
*   💻 **Framework:** React & TypeScript (with `@tanstack/react-router`)
*   ✨ **Icons:** Lucide React
*   📦 **Version Control & Hosting:** GitHub
