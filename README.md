# Atelier AI — Workplace Productivity Assistant

### Project Overview
Atelier AI is an elegant, single-platform workspace assistant designed to automate daily professional tasks. Built as a high-end luxury digital suite, the platform bridges modern user experience with workplace automation to handle communication, organization, and technical research seamlessly.

### Key Features Fully Implemented
1. **Smart Email Generator:** Composes tailored text block correspondence across multiple selectable professional tones.
2. **Meeting Notes Summarizer:** Distills long transcripts into distinct executive summaries, key decisions, and deadline tracking.
3. **AI Task Planner:** Transforms random lists into clean, interactive, chronological daily agendas.
4. **AI Research Assistant:** Moves from questions to strategic, boardroom-ready topic analysis.
5. **AI Concierge (Chatbot):** A conversational workspace partner for interactive, real-time queries.

### Tools & Tech Stack Used
* **Frontend Architecture & Generation:** Lovable AI
* **Framework:** React & TypeScript (with @tanstack/react-router)
* **Icons:** Lucide React
* **Version Control & Hosting:** GitHub
